# Legacy Minor-Illness Archive

This archive documents the pre-`pcmi` minor-illness system.

## Status

- `primary-care minor-illness` is now the canonical unified resource.
- The older `minor-ailments/community-pharmacy` and `minor-ailments/gp-advice` folders are legacy source material.
- The old `micp-*` and `migp-*` map families are retained only for backward compatibility and historical reference.

## Why the legacy folders remain in place

The original folder paths are still referenced by local extraction and authoring tooling, so the files remain readable in their existing locations even though they are now archived in status.
