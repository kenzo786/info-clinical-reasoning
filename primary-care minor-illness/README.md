# Primary-Care Minor-Illness

This folder is the staging area for a single primary-care minor-illness knowledgebase built from the legacy `community-pharmacy` and `gp-advice` resources.

Principles:

- `reference/` contains the canonical source files.
- `markmaps/` contains shorter staged maps derived from the canonical source files.
- `docs/` contains the scaffold, tracker, and working notes.
- The staged resource is non-live for now.
- Live `micp-*` and `migp-*` files in `clinicalreasoning-maps/src/md` are not changed during this build phase.

Batch 1 topics:

- acute cough
- headache
- back pain
- emergency hormonal contraception
- vaginal thrush
